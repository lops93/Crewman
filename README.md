<h1 align="center">Crewman</h1>


## installation

- clone the project

## api folder
- composer create-project command
- setup the .env file
- ./vendor/bin/sail up command to containerize the project
- ./vendor/bin/sail artisan migrate --seed command to run migrations and seeders


## client folder
- npm install
- quasar dev
